# Visual NCE

**项目愿景**：利用 Google Nano Banana Pro 的 AI 绘图能力，重构经典教材《新概念英语》（NCE 1-4），打造一个沉浸式的、听觉和视觉驱动的静态点读学习应用。

## 核心技术栈

- **Frontend**: Vue 3 + Vite
- **Data**: Static JSON files
- **AI**: Nano Banana Pro (via Antigravity / Gemini-cli)
- **Deployment**: Vercel (Hobby)

## 🚀 任务开发路线图 (Phase 1: MVP)
本项目采取原子化开发模式，每一步必须达成一个可运行的最小效果。

### 第一阶段：基础设施与数据建模
- [x] **Task 1.1**: 初始化 Vue 3 + Vite 项目。
  - **效果**：完成项目脚手架，能看到 Vite 欢迎页。
- [x] **Task 1.2**: 设计并创建第一课的数据协议 `src/data/lessons/l1.json`。
  - **效果**：包含音频 URL、文本片段、时间戳占位和图片路径。

### 第二阶段：AI 视觉预生产 (核心成本优化)

### 第四阶段：静态部署与运营

## 🛠 开发指南 (给 Gemini-cli 的指令)
1. **增量开发**：每次只执行一个 Task，完成后更新此 README 中的 `[ ]` 为 `[x]`。
2. **资源优先**：生成的图片必须使用 `media_resolution_high` 设置以保证清晰度。
3. **保持静态**：严禁引入 Spring Boot 或任何需要运行时数据库的逻辑，所有状态存储在 `LocalStorage` 中。
